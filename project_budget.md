# Project Budget Estimate

## Inputs Provided
- Material cost: RM54,000
- Software development effort: 240 hours
- Miscellaneous buffer: 1%

## Estimation Assumptions
- FX rate for normalization: USD1 = RM4.27 (as of 22 May 2025)
- Development labor rate: USD150 per 8 hours
- Installation labor effort: 40 hours
- Installation labor rate: USD150 per 8 hours
- Pre-Sales Cost: RM6,500 (recommended baseline)
- Installation travel: 4 site trips at RM500/trip
- Shipment: 2% of hardware/material cost
- Custom duty/tax: 6% of hardware/material cost
- Miscellaneous buffer (1%): applied to hardware/material cost

## Budget by Category
| Category | Basis | Amount (RM) |
|---|---:|---:|
| Hardware & Material | Given | 54,000 |
| Development Labor Cost | 240 hours / 8 x USD150 x RM4.27/USD | 19,215 |
| Installation Labor Cost | 40 hours / 8 x USD150 x RM4.27/USD | 3,202.50 |
| Pre-Sales Cost | Fixed allowance | 6,500 |
| Installation Travel Cost | 4 trips x RM500 | 2,000 |
| Shipment | 2% x RM54,000 | 1,080 |
| Custom Duty/Tax | 6% x RM54,000 | 3,240 |
| Miscellaneous | 1% x RM54,000 | 540 |

## Labor Cost Calculation
- Development labor: 240 hours ÷ 8 × USD150 = USD4,500 x RM4.27 = RM19,215.00
- Installation labor: 40 hours ÷ 8 × USD150 = USD750 x RM4.27 = RM3,202.50

## Total Project Budget (Estimated)
- Hardware & Material (RM54,000) + Development Labor (RM19,215.00) + Installation Labor (RM3,202.50) + Pre-Sales (RM6,500) + Installation Travel (RM2,000) + Shipment (RM1,080) + Custom Duty/Tax (RM3,240) + Miscellaneous (RM540)
- Total = RM89,777.50

## Notes
- Labor rates are based on USD150 per 8 hours for both development and installation.
- Pre-sales budget is set as a fixed planning allowance at RM6,500.
- FX normalization uses USD1 = RM4.27.
